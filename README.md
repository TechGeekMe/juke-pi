# juke-pi
A web-app used to play music at gatherings. Users can upload the songs of their choice and depending on the number of upvotes the song gets by other users, the playlist is sorted and played accordingly.

### Technologies used
- [Node.js](https://nodejs.org/)
- [MongoDB](https://www.mongodb.com/)
- [Music Player Daemon](https://www.musicpd.org/)
- [Polymer](https://www.polymer-project.org/)
- [Express](http://expressjs.com)
